# Code of Conduct

misfire is a small, open-source project. We want contributing to be productive and welcoming for everyone, regardless of experience level.

## Our Standards

- Be respectful and constructive. Disagree about ideas, not people.
- Welcome contributors of all experience levels — questions are fine, "obvious" is not a useful word.
- Focus on the work: the rule, the trace, the test, the diff. Critique the idea, not the individual.
- Assume good intent. Most friction is a misunderstanding, not malice.

## Enforcement

Maintainers may edit, remove, or reject comments, commits, code, issues, and pull requests that do not align with this Code of Conduct. Repeated or severe violations may result in a temporary or permanent ban from the project.

## Reporting

- **Public concerns:** open a GitHub issue labeled `conduct`.
- **Private or sensitive reports:** use [GitHub Security Advisories](https://github.com/ek33450505/misfire/security/advisories) or email edward.kubiak.dev@gmail.com.

Reports are handled discreetly and in good faith.

## Reference

This Code of Conduct follows the spirit of the [Contributor Covenant](https://www.contributor-covenant.org), v2.1. See the linked text for the full community standard.
